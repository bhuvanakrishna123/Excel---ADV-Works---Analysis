# Excel Adventure Works Sales Analysis

## Project Overview
This project analyzes Adventure Works sales data using Microsoft Excel. The dashboard provides insights into sales performance, customers, products, and business trends.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting

## Dashboard Features
- Total Sales
- Total Profit
- Total Orders
- Monthly Sales Trend
- Top Products
- Top Customers
- Regional Sales Analysis

## Key Insights
- Identified top-performing products.
- Analyzed monthly sales trends.
- Compared regional sales performance.
- Found high-value customers.

## Skills Demonstrated
- Data Cleaning
- Data Analysis
- Dashboard Development
- Business Reporting
- Data Visualization
